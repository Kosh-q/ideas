Десткая стекловата
Какая она, детская стекловата?
Обычная (строительная) стекловата — это колючие стекловолокна, которые вредят коже и лёгким.
А ваша, детская, наверняка:

Мягкая, как пух, но при этом держит форму.

Не колется — сделана из современных безопасных волокон (например, из ПЭТ-волокна или силиконизированного полиэстера).

Гипоаллергенная, её можно мять, рвать, клеить, красить.

Продаётся в ярких упаковках с картинками и называется, скажем, «Стекловата-Невидимка» или «ВатаФайбер Креатив».

Что с ней делать?
Если это творческий материал, то он может заменить:

Обычную вату (которая сваливается)

Синтепон (который скользкий)

Флок (который сыпется)

Из детской стекловаты можно:

Лепить объёмные фигурки (смешивая с клейстером)

Делать «шерсть» для игрушек

Создавать облака, сугробы, бороду Деда Мороза
